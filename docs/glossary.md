# Bilingual terminology glossary (HU ↔ EN)

Controlled vocabulary for all policy deliverables. The `translator` MUST use
these equivalents; the `translation_checker` enforces them. Additions require
a new row with a note; changing an existing equivalent is a system change and
must appear in a round's `revised_workflow.md`.
